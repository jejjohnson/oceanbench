Scaling Domains#

This demo notebook showcases some key functions for scaling coordinates. We often need to use semantic coordinate values that are actually useful for computation. So for example, lat/lon coordinates are often in the spherical domain. But for many physical problems, we need them in meters. Similarly with time coordinates: we often have np.datetime64 coordinate values but we need them in seconds.

Note: There are some more meaningful rescaling that involve the mean and variance (StandardScaler) or specific min/max values (MinMaxScaler) but this is an arbitrary transformation that isn’t specific to coordinate values.
